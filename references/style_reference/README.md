# 固定视觉参考图

本目录存放图片生成阶段使用的固定风格参考：

- `PPT模板.png`：标题区、开放内容区和字幕安全区。
- `PPT示例.png`：手绘风格、图标、标注、分栏和视觉密度。

## 当前规则

- 每页生成一张完整的 1920×1080 位图。
- 图片外围背景必须为纯白色 `#FFFFFF`。
- PPT 主体内容保持在 `y < 930`。
- 标题、正文、图标、箭头、框线和总结条都由图片生成模型绘制。
- Remotion 不重新绘制 PPT 主体，只显示完整页或手工 Mask 生成的 PNG reveal 图层。
- 最终视频背景色由项目设置决定，默认 `#FEFDF9`。

更换参考图时必须同步检查 `config/style_tokens.yaml` 和生图提示词。
